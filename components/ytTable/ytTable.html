# ytTable基于el-table封装的表格组件

# 使用场景

有表格的页面，不适用于树形结构的表格

# 实例属性

属性名 默认值 类型 描述 是否必填
tableData [] Array 表格数据(和el-table的data属性一致) true
tableDataColumn [] Array 表头数据 true
tableConfig {} Object 表格的配置项 true
rowStyle {} Object 表格行样式
headerRowStyle {} Object 表头样式
height "45vh" String 表格的高度
idName "id" String 表格行的唯一标识
paginationConfig {} Object 分页参数
expandOnly false Boolean 是否只允许展开一行

# tableConfig 属性

属性名 默认值 类型 描述 是否必填
loading false Boolean 是否有loading
stripe true Boolean 是否为斑马纹
border true Boolean 是否带有纵向边框
resizable true Boolean 对应列是否可以通过拖动改变宽度（需要在 el-table 上设置 border 属性为真）
isAddIndex true Boolean 是否将分页后的序号进行累加（如为true则分页后有10条数据，第二页第一条的数据序号为11）

# rowStyle 属性

属性名 默认值 类型 描述 是否必填
background "#fafafa" String 表格单行的背景色
boxShadow "inset 0px -1px 0px 0px #EEEEEE" String 表格单行的边框
fontFamily "MicrosoftYaHei" String 表格单行的字体
color "#606266" String 表格单行的字体颜色
height "24px" String 表格单行的高度

# headerRowStyle 属性

属性名 默认值 类型 描述 是否必填
color "#606266" String 表格单行的字体颜色

# paginationConfig 属性

属性名 默认值 类型 描述 是否必填
pageNum 1 Number 分页的参数（当前页码）
pageSize 10 Number 分页的参数（每页展示的数量）

# 事件

事件名 参数 描述
handleSelectionChange {ids: ids, // 选中行组成的ids数组<br/>selection, // 选中行的数据} 获取选中行组成的ids数组和选中行的数据
handleCellDbClick {} 表格单行双击事件,获取选中行的数据
handleResetSelection 表格多选框全选/全部取消选中

    }
  },

    /**
     * @Event 表格多选框全部取消选中
     * @description:

    handleResetSelection() {
      this.$refs.ytTable.handleResetSelection(

      this.isSelectAll = !this.isSelectAll;

    /**

     * @author: mhf
     * @time: 2023-11-24 23:57:49
     **/

      console.log(this.$refs.ytTable.ids)
    }
  }
}
</script>
